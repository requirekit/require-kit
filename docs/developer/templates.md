# Templates

Templates and patterns for extending RequireKit.

## Available Templates

### Implementation Plan Template
Template for documenting implementation plans.

**Location:** `docs/templates/implementation-plan-template.md`

## Creating Custom Templates

RequireKit's markdown-based approach makes it easy to create custom templates:

1. Create markdown file with frontmatter
2. Define required fields
3. Add section structure
4. Document usage

## Template Best Practices

- Use YAML frontmatter for metadata
- Include clear section headers
- Provide examples in comments
- Document required vs optional fields

For template files, see the `docs/templates/` directory.
